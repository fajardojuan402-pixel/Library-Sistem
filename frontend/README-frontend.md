# Frontend React - Biblioteca

## Requisitos
- Node.js 16+
- npm

## Instalación & ejecución
1. En la carpeta del frontend:

```bash
npm install
npm start
```

2. Si tu backend Laravel corre en otra URL, crea `.env` con:

```
REACT_APP_API_URL=http://127.0.0.1:8000
```

La app abre en http://localhost:3000
